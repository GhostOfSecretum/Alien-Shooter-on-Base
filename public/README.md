# Изображения для Base Mini App

Эта папка должна содержать следующие изображения:

## Необходимые изображения

1. **icon.png** (512x512px)
   - Иконка приложения
   - Формат: PNG
   - Размер: 512x512 пикселей

2. **splash.png**
   - Изображение для splash screen при запуске
   - Рекомендуемый размер: 1080x1920px (портретная ориентация)

3. **hero.png**
   - Главное изображение приложения
   - Рекомендуемый размер: 1200x630px

4. **screenshot-portrait.png**
   - Скриншот приложения в портретной ориентации
   - Рекомендуемый размер: 1080x1920px

5. **og-image.png** (1200x630px)
   - Изображение для Open Graph (социальные сети)
   - Формат: PNG или JPG
   - Размер: 1200x630 пикселей

## Временные placeholder изображения

Вы можете создать простые placeholder изображения для тестирования, используя любой графический редактор или онлайн-генератор placeholder изображений.

## Рекомендации

- Используйте высокое качество изображений
- Оптимизируйте размер файлов для быстрой загрузки
- Убедитесь, что изображения соответствуют брендингу вашего приложения
